# Week 8 — Deployment + Capstone Ship

**Why:** "Ship real agents" means it runs somewhere other than your laptop.

## Learn (~3 hrs)
- Packaging the agent as a service
- A **FastAPI** endpoint (or a serverless function) wrapping the capstone
- Secrets / env management in production (no keys in code)
- Minimal monitoring + structured logging in production

## Build (~2 hrs)
- Wrap the capstone behind an API endpoint
- Deploy it (a free-tier host, container, or serverless platform)
- Write the README + a short demo (GIF or transcript)

## Deliverable
- Deployed capstone agent reachable via an endpoint
- `README` with: architecture diagram, costs, eval results (from Week 7), known limitations

## Self-check
A second person can hit your endpoint and get a useful, grounded result.
