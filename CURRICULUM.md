# AI Engineering & Agents — 8-Week Curriculum

A project-spine curriculum for resharpening coding skills with the goal of **shipping real agents**.
~5 hrs/week × 8 weeks. You build one agent across the whole course; each week adds a capability.

**Capstone (the spine):** Product Research & Prioritization Agent — ingests unstructured inputs
(feedback, notes, docs), retrieves from a knowledge base, reasons with tools, and outputs a
prioritized recommendation / draft PR-FAQ. _(Confirm capstone choice at kickoff — see RESOURCES.md / plan.)_

How to use this: each week lives in its own folder with a `README.md`. Check off tasks as you go.
After Week 1's diagnostic, we recalibrate the pace.

## Weekly format

Each week ≈ 5 hrs = ~3 hrs learning + ~2 hrs building, and ends with:
- **Deliverable** — committed to the repo.
- **Self-check** — how you know it worked.

## Calibration checkpoint

After Week 1's diagnostic, revisit the pace:
- Python/tooling solid → compress Week 1, pull Week 2 forward.
- Rustier than expected → add a remedial half-week before Week 2.
